# REST-API

Este projeto usa:

* Node
* Swagger
* MySQL
* bcrypt
* express
* JWT (JasonWebToken)

## Pré-requisitos

* [Git](https://git-scm.com/)
* [Node](http://nodejs.org/)
* [MySQL](https://www.mysql.com/)


## Respositório na núvem

* [GitHub](https://github.com/rickscombr/api-rest.git)

## Instalação

```sh
$ npm install
```

## Executando Local (DEV)

```sh
$ npm start
```

## Deploy para produção

```sh
$ npm build
```
